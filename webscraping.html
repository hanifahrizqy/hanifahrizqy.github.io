Database Tiktokstar

Scraping data from Famous TiktokStar website on 3rd August 2021. And get
the database and the visualization in google sheets.

Goals

This information is useful for digital marketing companies to get to know who is in the top chart among the other tiktok celebrities with jupyter python. And analyze them in google spreadsheet.

Data Description

The dataset is about their biography such as birthplace, family life, with who he/she is associated with, zodiac sign,etc. You can look further in this Database. And I took only 48 famous tiktok star because the website only provided for top 48 famous tiktok star at the moment.

Details of project

First, this website i used on 3rd August 2021. And maybe for now you open up the website it is completly different person because it changed entirely arbitrary. The code for scraping the website in this github. After getting the data I was seeking, I emerged the data into database and visualize it.
