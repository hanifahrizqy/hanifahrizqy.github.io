Text Processing of Twitter Data

Using Word cloud which filled with words in different size, which represent
frequency or the importence of each word.

Goals

Get to know the importance or the frequency of each word who mentioned twitter account of @gopayindonesia with python programming language.

Data Description

Got this dataset from my lecturer and processing with python. This dataset based on twitter user who mentioned @gopayindonesia and get more than 1182 rows in CSV file. Just click here for download the dataset.

Details of project

First of all, for text preprocessing I used sastrawi and nltk packages. After we cleaned the data, we began to turn all of the words into word cloud with Indonesian stopwords note.
